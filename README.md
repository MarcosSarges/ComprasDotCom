This is a new [**React Native**](https://reactnative.dev) project, bootstrapped using [`@react-native-community/cli`](https://github.com/react-native-community/cli).

# Getting Started

## Step 1: Start the Metro Server

```bash
# using npm
npm start

# OR using Yarn
yarn start
```

## Step 2: Start your Application

### For Android

```bash
# using npm
npm run android

# OR using Yarn
yarn android
```

### For iOS

```bash
# using npm
npm run ios

# OR using Yarn
yarn ios
```

### Requirements

To run this React Native project, you will need to have the following software installed on your machine:

- Node.js version 20.x or above
- Java 17.x or above
- Xcode 15.x or above

Make sure to have these requirements fulfilled before proceeding with the next steps.
